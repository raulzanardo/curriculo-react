## English Version

### About

This is my personal portfolio and interactive resume, designed to showcase my experience as a Full-Stack Developer specializing in Flutter, React, and NestJS. The project demonstrates my skills in web and mobile development, UI/UX design, and modern development practices.

### Features

- Responsive Design: Optimized for desktop, tablet, and mobile devices
- Multi-language Support: Available in Portuguese (BR) and English (US)
- Dark/Light Theme: Toggle between dark and light modes for comfortable viewing
- Interactive Portfolio: Detailed case studies of my projects, including MyOrbit, 24Pet Shelter, and CoPal
- Modern UI: Built with React and Tailwind CSS
- Clean Architecture: Well-organized codebase following React best practices

### Technologies

Framework & Language:

- React 19
- TypeScript 5.9+

Key Packages:

- React Router 7 - Client-side routing
- Tailwind CSS 4.1+ - Utility-first CSS framework
- Lucide React - Icon library
- Vite - Build tool and development server

Platform Support:

- Web (Chrome, Safari, Firefox, Edge)
